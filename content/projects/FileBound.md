# FileBound

###### Website to store and share files with an encrypted and time-bounded URL.

## Features
1. Share Link to download files/folders
2. Link Expiry after custom number of downloads
3. Link Expiry after time
4. End-to-End Encrypted URLs with key attached as a slug to the final URL
