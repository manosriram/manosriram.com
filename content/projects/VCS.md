# Mini Version Control

###### Control-System which helps file tracking and storing different versions of files.

## Features
1. Uses SHA1 hash to track file changes.
2. Commits to save current state of directory
3. Stage/Unstage files
4. View content of the file, given it's hash
5. View history of commits with timestamps
6. View the tree of a specific commit
